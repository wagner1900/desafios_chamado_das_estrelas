Desafio Python Interativo

Bem-vindo à seção interativa dos desafios de O Chamado das Estrelas! Sua tarefa é decifrar a mensagem criptografada "frphvdu" usando a cifra de César com um deslocamento de 3. Ao executar o código corretamente, o resultado (por exemplo, "codigo") servirá como chave para desbloquear novas páginas do livro.

Você pode editar o código, mas lembre-se: o código não pode ultrapassar 1000 caracteres. Faça modificações com cuidado!
